## Landing Page – Site de Restaurante

## Sobre o projeto

Este projeto é uma landing page de restaurante, desenvolvida com foco em design moderno, organização de código e interatividade com o usuário.
O site apresenta seções como página inicial, cardápio, depoimentos e contato, proporcionando uma navegação clara e intuitiva.

## Tecnologias utilizadas

HTML5

CSS3

JavaScript

jQuery

Font Awesome

ScrollReveal

## Funcionalidades

Layout responsivo

Menu de navegação interativo

Animações ao rolar a página

Destaque automático da seção ativa no menu

Estrutura organizada com CSS separado por seções
